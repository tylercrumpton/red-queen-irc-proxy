I can give you the code, but first you must do something for me.

In order to use my IRC REST proxy interface, you must first acquire an API key from the server admin.

After acquring a key, simply make an HTTP GET request as follows:

curl --data '{"message":"My message", "channel":"##rqtest", "isaction":false, "key":"13371234"}' http://www.myredqueenurl.com/relay --header "Content-Type:application/json"

Features

  • Help page with usage information
  • Allows both messages and action to be sent via REST API
  • Allows HTTPS for secure API key transmission
  • Can use ##rqtest channel for testing with a sample key
  • User/application authentication via API keys
